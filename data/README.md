# Data Directory

Input files required for the analysis (not tracked by Git):

| File | Description |
|------|-------------|
| `Alldata2016-2018 3sd NA_FINAL_NO_ISSUES.csv` | Multi-year phenotype data (282 varieties × 3 years × 2 systems × 2 reps) |
| `sharma_genotype_reduced_recoded.csv` | SNP dosage data for population structure |
| `sharma_genotype_GWAS_poly.csv` | Genotype matrix formatted for GWASpoly |
| `sharma_SNP_positions.xlsx` | Marker positions and chromosome annotations |
| `K3.outfile` | STRUCTURE output (K=3) |

## Source
Sharma, S.K. et al. (2018). G3, 8(10), 3185-3202. doi:10.1534/g3.118.200377
